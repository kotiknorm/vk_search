Проект реализован в android studio
использованы vksdk_library и actionbarsherlock
сборка через gradle


Список известных проблем:
1. Не реализован поиск по городу
2. При авторизации через приложение экран с ошибкой из которого можно уйти в приложение
3. При входе в активити параметров все параметры сбрасываются
4. Когда не приходит photo_200, то вместо аватарки - белый экран
5. Добавить в класс People id пользователя



Список предложений по улучшению:
1. Сделать поисковой запрос по стране с названием страны а не ее id и всю логику по этому моменту перенести на сервер.
2. Сделать это же с городом.
3. Непонятный момент с пустым запросом - возвращает друзей друзей, при добавлении любого поля - уже поисковой запрос по всем пользователям. Можно сделать какое нибудь поле обязателньое для этого.



Для использования кода как отдельного модуля, достаточно получить listView в активити PeopleListActivity и названчить ему обработчика нажатия (например для открытия страницы пользователя)
В каждой View списка хранится экземпляр класса People;
